# Controle de Gastos para iPhone

Esta pasta e a versao PWA do app. Ela nao precisa de backend e salva os gastos no proprio iPhone.

## Como colocar no iPhone

1. Publique todos os arquivos desta pasta em um site com HTTPS.
2. Abra o link no Safari do iPhone.
3. Toque no botao de compartilhar.
4. Toque em `Adicionar a Tela de Inicio`.
5. Confirme o nome `Gastos`.

Depois disso, o app aparece como icone na tela inicial.

## Onde os dados ficam

Os gastos ficam salvos no armazenamento local do Safari/iPhone. Se o site for apagado dos dados do Safari, os registros tambem podem ser apagados.

## Jeito mais simples de publicar

Use um servico de hospedagem estatica com HTTPS, como Netlify, Vercel, Cloudflare Pages ou GitHub Pages. Basta enviar/subir os arquivos desta pasta.
